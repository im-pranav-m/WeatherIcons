# WeatherIcons 
A modern and lightweight weather icon pack designed for dashboards, mobile apps, websites, IoT displays, and embedded systems.

This repository contains weather icons available in multiple resolutions for maximum flexibility and performance.


#### Available in multiple sizes
- 512×512
- 256×256
- 128×128
- 64×64
- 32×32
- 24×24
- 16×16

Right now, It doesn't contain much Icons, But will be updated later. You may also contribute by adding Icon packs, the running the IR.py in a folder containing 512x512 icons, 
the script will resize the images to different sizes and will be added to respective folders. the PY script uses pillow library.
